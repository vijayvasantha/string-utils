# String Manipulation Utilities

A Python script with reusable functions for common string operations.

## Features
- **Capitalize Words:** Capitalizes the first letter of each word in a sentence.
- **Palindrome Checker:** Checks if a word or phrase is a palindrome.
- **Random Password Generator:** Generates a random secure password.
